# React et Webpack

Lancement du projet

```bash
npm start
```

Build du projet

```bash
npm run build
```

Nettoyage du projet, supprime le dossier `/dist`

```bash
npm run clean
```

Gros nettoyage du projet, supprime les dossiers `/dist`, `node_modules` et les fichiers `package-lock.json` et `yarn-lock`

```bash
npm run clean:all
```
